Bootstrap Alerts Documentation

Basic Alert Variants

Bootstrap provides eight different alert styles to convey different types of messages:

A primary alert - Often used for main information messages
A secondary alert - Used for supplementary information
A success alert - Indicates a successful or positive action
A danger alert - Warns about dangerous or negative actions
A warning alert - Used for warning messages that need attention
An info alert - Provides general information to the user
A light alert - Subtle alert with light background
A dark alert - Subtle alert with dark background
