BOOTSTRAP CARDS
===============

Bootstrap cards are flexible and extensible content containers that provide a way to organize and display content in a clean and organized manner.

1. BASIC STRUCTURE
-----------------
A basic card includes these main components:
• card: The main container class
• card-header: Optional header section
• card-body: Main content area
• card-footer: Optional footer section
• card-title: Title of the card
• card-text: Text content
• card-img-top/bottom: Images at top or bottom

2. COMMON CARD COMPONENTS
------------------------
Example HTML structures:

Basic Card:
<div class="card">
<div class="card-body">
    <h5 class="card-title">Card Title</h5>
    <p class="card-text">Some quick example text.</p>
    <a href="#" class="btn btn-primary">Go somewhere</a>
</div>
</div>

Card with Image:
<div class="card">
<img src="..." class="card-img-top" alt="...">
<div class="card-body">
    <h5 class="card-title">Card Title</h5>
    <p class="card-text">Card content here.</p>
</div>
</div>

3. CARD FEATURES
---------------
• Sizing: Can be sized using grid classes or custom CSS
• Text alignment: Use text-center, text-right, etc.
• Background colors: bg-primary, bg-success, etc.
• Border styling: border-primary, border-success, etc.

4. CARD LAYOUTS
--------------
• Card Groups: Group cards with equal height and width
• Card Decks: Similar width/height with gutter in between
• Card Columns: Masonry-like grid of cards

5. SPECIAL FEATURES
------------------
• List groups in cards
• Navigation in cards
• Header and footer variants
• Horizontal cards
• Card overlays with images

6. RESPONSIVE BEHAVIOR
--------------------
Cards can be made responsive using Bootstrap's grid system:

<div class="row">
<div class="col-sm-6">
    <div class="card">
    <!-- Card content -->
    </div>
</div>
</div>

7. UTILITY CLASSES
-----------------
• w-100: Full width
• h-100: Full height
• shadow: Add shadows
• rounded: Control border radius

COMMON USE CASES
---------------
Cards are commonly used for:
• Product displays
• User profiles
• Feature presentations
• Blog post previews
• Dashboard widgets

Note: Always ensure to include the necessary Bootstrap CSS and JS files in your project to use cards effectively.
